# ObservatorioRaioSeguro

Dashboard institucional (Vite + React) para leitura de sinistros de transito do Programa RAIO Seguro - CPRaio.

Link (GitHub Pages):
https://abraaorosal.github.io/ObservatorioRaioSeguro/

## Rodar local

```bash
npm install
npm run dev
```

## Dados

O dashboard le `public/sinistro_transito.json`. Para usar a base real, substitua esse arquivo pelo JSON institucional.
